# Tic-tac-toe Game 

Tic-tac-toe Game created using React 

## Getting Started 

* Run the following 

``` 
npm install 
``` 

``` 
npm start 
``` 

* The app can then be accessed at [http://localhost:3000/](http://localhost:3000/) 

## Features 

* Click to move
* Displays current player and game status
* Highlights winning squares
* Jump to any previous move
